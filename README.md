# crystaledge

Pure Crystal vector math library (**WIP**)


## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  crystaledge:
    github: unn4m3d/crystaledge
```


## Usage


```crystal
require "crystaledge"
```
